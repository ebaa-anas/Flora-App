Flora 🌿 - Premium Plant E-Commerce & Care
Flora is a professional, full-stack Android application built with Jetpack Compose and Supabase. It provides users with a seamless experience to discover plants, manage their own greenhouse collection, and track orders in real-time.

✨ Main Features
Full Authentication: Secure Sign-up and Login system powered by Supabase Auth.

Smart Plant Discovery: Browse plants by categories (Indoor/Outdoor) with a functional search bar.

Detailed Care Guides: Essential care info (Light, Water, Temp) and "View in AR" options.

Dynamic Shopping Cart: Real-time price calculations for plants and premium pot upgrades.

Real-time Order Tracking: A visual step-by-step timeline for active deliveries.

Personal Greenhouse: Dashboard to manage owned plants and daily hydration tasks.

UI/UX Excellence: Fully responsive design with Dark Mode and Light Mode support.

🛠️ Tech Stack
UI: Jetpack Compose & Material 3

Backend: Supabase (Auth, Database, RLS)

Image Loading: Coil

Data: Kotlin Serialization & Ktor
